## milestone 2

회원 가입, 로그인, 추후 회원별 기능 이용을 위해

회원은 이메일과 비밀번호를 입력하여 가입한다.
토큰을 받으려면 이메일과 비밀번호를 보내야 하며, 가입한 이메일과 비밀번호가 일치하면 토큰이 발급된다.
토큰은 JWT 를 사용하도록 한다.
관리자는 회원을 조회 추가 수정 삭제 할 수 있다.
